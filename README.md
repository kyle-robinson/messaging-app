# Messaging Application

A multithreaded online messaging application that utilises a client-server architecture to allow for devices to send messages across the local network.

## List of Features

- [x] TCP/IP and UDP
- [x] Global Messaging
- [x] Private Messaging
- [x] Cryptography
- [x] Login System
- [x] Muting System
- [x] Client/Friend Lists
